格子中输出 （作业/来源于：2015年蓝桥杯B组）

填空的地方就是输出字符串的地方

而“%* s”：在scanf里用修饰符，是起到过滤读入的作用；在printf中的含义完全不同，是代表设置域宽的意思， 修饰符用来灵活的控制域宽

由题意可知要字符串要居中，去掉首尾的“|”再分左边和右边

域宽即为“(width - strlen(buf) ) / 2 - 1” 或 “(width - strlen(buf) - 2 / 2)”

但是题目说了要求字符串在水平、垂直两个方向上都居中。字符串太长，就截断。如果不能恰好居中，可以稍稍偏左或者偏上一点。

当字符串个数为奇数时，以上答案得出的图案并没有对齐

所以两边的域宽不能相等， 字符串为奇数个时应该左边的域宽比右边的小
