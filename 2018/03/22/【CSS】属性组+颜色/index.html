CSS颜色

CSS 的颜色是通过对红、绿和蓝光的组合来显示的。

使用组合了红绿蓝颜色值 (RGB) 的十六进制 (hex) 表示法进行定义。

对光源进行设置的最低值可以是 0（十六进制 00）。最高值是 255（十六进制 FF）。

十六进制值使用三个双位数来编写，并以 # 符号开头。

CSS合法颜色值

可以用以下方法来规定 CSS 中的颜色：

  • 十六进制色：所有浏览器都支持十六进制颜色值。十六进制颜色是这样规定的：#RRGGBB，其中的 RR（红色）、 GG（绿色）、BB（蓝色）十六进制整数规定了颜色的成分。所有值必须介于 0 与 FF 之间。
  • RGB 颜色：所有浏览器都支持 RGB 颜色值。可以是介于 0 与 255 之间的整数，或者是百分比值（从 0% 到 100%）
  • RGBA 颜色：支持以下浏览器：IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+。带有一个 alpha 通道 - 它规定了对 象的不透明度。alpha 参数是介于 0.0（完全透明）与 1.0（完全不透明）的数字。
  • HSL 颜色：支持以下浏览器：IE9+、Firefox、Chrome、Safari 以及 Opera 10+。
  • HSLA 颜色：支持以下浏览器：IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+。
  • 预定义/跨浏览器颜色名

CSS颜色名
